# What is Vadalog?

This is a short user manual for **Vadalog**, the language underpinning Prometheux's reasoning engine.

As a language of the Datalog +/- family, Vadalog is declarative and highly expressive.

Because it's so expressive, _**all**_ SQL and Cypher queries can be expressed simply in Vadalog.

With Vadalog, you can express recursive statements and represent facts not in your database as variables to be deduced.
