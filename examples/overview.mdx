Explore practical Vadalog programs across two categories:

## Patterns

Fundamental programming patterns in Vadalog — recursion, graph analytics, and state machines. These examples illustrate core language features and are a good starting point for understanding how Vadalog rules compose.

- [Graph Analytics](/examples/patterns/graph-analytics) — Transitive closure, degree centrality, and community detection using pure Vadalog rules
- [Recursion](/examples/patterns/recursion) — Base and recursive cases, with a worked transitive closure example
- [State Machine](/examples/patterns/state-machine) — Modeling event-driven state transitions with recursive rules

## Finance

Real-world financial use cases drawn from regulatory and payment processing workflows.

- [Banking Supervision](/examples/finance/banking-supervision) — Company control structures and close-link detection for regulatory analysis
- [Card Clearing Migration](/examples/finance/card-clearing-migration) — Migrating mainframe COBOL card clearing pipelines to Prometheux
- [Non-Performing Loans](/examples/finance/non-performing-loans) — Cascading default propagation across interconnected financial entities
