### What's the difference between Prometheux and a graph database?

Prometheux is an ontology-native reasoning engine that goes beyond simple graph storage. It provides declarative rule-based reasoning, recursive query capabilities, and semantic understanding of your data.

### Can I use Prometheux with my existing database?

Yes. Prometheux integrates with a wide range of data sources including SQL databases, NoSQL stores, data lakes, and REST APIs. See the [Data Sources](/vadalog/data-sources) guide for the full list.

### What programming languages are supported?

Prometheux provides SDKs for Python, JavaScript/TypeScript, Java, and Go. The core reasoning is done in Vadalog, the declarative logic programming language at the heart of Prometheux.
