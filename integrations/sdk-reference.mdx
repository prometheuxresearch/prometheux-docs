This page is the complete index of the `prometheux_chain` Python SDK. Every
function is re-exported at the top level, so you call it as `px.<function>(...)`:

```python
import prometheux_chain as px

project_id = px.save_project(project_name="my_project")
```

Each function is a thin wrapper around a REST endpoint — follow the **Endpoint**
link to the full reference (parameters, request body, and response shape). Path
segments like `{project_id}` are supplied as function arguments.

<Note>
A few functions are **generators** that stream events as they arrive
(newline-delimited JSON or a WebSocket). They are flagged
<Icon icon="bolt" /> below — iterate over the return value rather than awaiting a
single result. See [Streaming responses](/integrations/python-sdk#streaming-responses).
</Note>

## Config

The `config` object reads and writes local SDK settings (token, backend URL, LLM
keys). See [Config](/integrations/config) for the full list of keys.

| Function | Description |
|---|---|
| `config.get(key, default=None)` | Read a local config value. |
| `config.set(key, value)` | Set a local config value. |
| `config.update_config(updates)` | Set multiple config values at once. |
