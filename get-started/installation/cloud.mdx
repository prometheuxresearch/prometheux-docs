> The fastest way to start with Prometheux — nothing to install.

## Overview

Prometheux Cloud is a fully managed platform hosted by the Prometheux team.
There is no infrastructure to provision, no software to install, and no
maintenance required on your side. You get immediate access to the full
platform including the UI, JupyterLab environment, and APIs.

## Prerequisites

- A Prometheux account (request access below)

## Quick Start

1. **Request access** — [Book an intro call](https://calendly.com/prometheux/prometheux-intro?utm_source=docs) or contact the Prometheux team to obtain your credentials.

2. **Sign in** — Navigate to [platform.prometheux.ai](https://platform.prometheux.ai/login) and log in with the credentials provided to you.

3. **Start building** — Once logged in, you have access to the full Prometheux platform: create ontologies, connect data sources, and run queries.

## What's Included

| Component | Description |
| --- | --- |
| **Platform UI** | Web-based interface for building and managing ontologies, data sources, and queries |
| **JupyterLab** | Integrated notebook environment with Python and Vadalog kernels |
| **REST API** | Programmatic access to the Prometheux engine |
| **Python SDK** | `pip install prometheux_chain` — see the [SDK documentation](/integrations/python-sdk) |
