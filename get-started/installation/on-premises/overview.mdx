Deploy Prometheux on infrastructure you control — whether a single VM on
AWS, Azure, or GCP, or a multi-node Yarn / Kubernetes cluster.

| Mode | Best For | Infrastructure | Effort |
| --- | --- | --- | --- |
| [**Single Node**](/get-started/installation/on-premises/single-node) | Quick setup on one VM | Docker on a Linux VM | Low |
| [**Cluster**](/get-started/installation/on-premises/cluster) | Distributed processing at scale | Yarn or Kubernetes cluster | High |

Both modes require credentials from the Prometheux team.
[Contact us](mailto:support@prometheux.co.uk) to get started.
